=======================================
ScreenPy Selenium: A ScreenPy Extension
=======================================

Not sure what ScreenPy is?
Check out `ScreenPy's documentation <https://screenpy-docs.readthedocs.io/en/latest/>`__ first!

``screenpy_selenium`` extends ScreenPy
by providing an Ability,
Actions,
Questions,
and Resolutions
which interact with `Selenium <https://www.selenium.dev/>`__.
This enables your Actors
to :class:`~screenpy_selenium.abilities.BrowseTheWeb`!
